# Novel Agent Product Brief

## 定位

`novel-agent` 是一个面向中文长篇类型小说创作者的 AI 写作助手。它不是一次性“帮我写一本书”的黑盒，而是一个能维护小说项目状态、持续生成章节、审稿并支持人工迭代的写作管线。

## MVP 用户

- 独立作者
- 网文作者
- 想用 AI 辅助搭建世界观、大纲和章节草稿的人

## MVP 场景

1. 输入一个灵感。
2. 生成 premise、世界观、角色卡和风格指南。
3. 生成主线和章节大纲。
4. 按章节生成草稿。
5. 对章节做一致性和文风审稿。

## 设计原则

- 人类作者保留最终控制权。
- 所有输出都保存为可读 Markdown/YAML。
- 默认不覆盖人工编辑过的章节。
- 先保证完整写作流程，再追求多 Agent 和 Web UI。
